# FUTURE_FS_03  
Jasmeet Singh – Pizza Website
A modern, interactive pizza ordering UI showcasing delicious pizzas with smooth animations, rotating dish slider, and visually appealing design.

Features

Hero section with pizza slices animation using GSAP

Smooth scroll-triggered animations for engaging UI

Rotating dish slider with next/previous controls

Dynamic background gradient transitions for each dish

Responsive design for desktop and mobile

Fixed “Order Now” button linking to ordering section

Technologies Used

HTML5

CSS3

JavaScript

GSAP (GreenSock Animation Platform)
